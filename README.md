# Slay the Spire RL Agent

This project trains a reinforcement learning agent capable of playing **Slay the Spire** using self-play, neural networks, and game simulation.

### Goals
- Build a custom environment wrapper for STS
- Train an RL agent (PPO or DQN)
- Experiment with deck-building decision strategies
- Visualize run statistics & learned policies

### Tech Stack
- Python
- PyTorch
- RL Algorithms (PPO, DQN)
- Custom STS game environment

### Status
Early prototype, environment exploration under development.

Heuristic vs RL Decision Boundary

Non-combat screens (card rewards, rest sites, map pathing, shop, events, boss relics) are currently handled by hand-coded heuristics. The RL policy only controls combat actions. Over time, these heuristic screens will migrate under full RL control as the policy matures and network capacity increases, eventually removing the heuristic layer entirely. The goal is a single end-to-end RL policy that handles every decision in the game.
